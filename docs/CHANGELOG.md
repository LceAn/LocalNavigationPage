# 更新日志 (CHANGELOG)

## [1.1.0] - 2026-03-19

### ✨ 新增功能

#### 多 URL 支持
- 一个网站可以配置多个 URL 地址
- 支持设置 URL 标签（如：本地网络、Tailscale、向日葵）
- 支持设置 URL 优先级
- 卡片右上角显示下拉按钮，可快速选择不同地址
- 设置界面中显示多 URL 信息（+N 个地址）

#### 外观与显示设置合并
- 将原来的"外观"和"显示设置"标签页合并为"外观与显示"
- 简化设置导航，从 6 个标签页减少到 5 个

#### 天气功能增强
- 支持配置多个城市显示天气
- 多城市时在天气后显示城市名称列表（如：shanghai / beijing / guangzhou）
- 单个城市时不显示城市名称，保持简洁

### 🔧 功能优化

#### 设置菜单优化
- 合并相关设置项，使界面更加简洁
- 优化设置分类逻辑，将时间、日期、天气等显示相关设置归为一类

#### 数据格式兼容
- 新的多 URL 格式使用 `urls` 数组存储
- 保持与旧格式 `url` 字段的兼容性
- 单个 URL 时自动使用旧格式，多个 URL 时使用新格式

### 📝 数据结构

#### 多 URL 数据格式
```json
{
    "links": [
        {
            "ID": 1,
            "name": "群晖 NAS",
            "urls": [
                {
                    "address": "https://192.168.2.100:5000",
                    "label": "本地网络",
                    "priority": 1
                },
                {
                    "address": "https://100.64.0.5:5000",
                    "label": "Tailscale",
                    "priority": 2
                }
            ],
            "category": "云服务",
            "thumbnail": "..."
        }
    ]
}
```

#### 单 URL 数据格式（兼容）
```json
{
    "links": [
        {
            "ID": 1,
            "name": "群晖 NAS",
            "url": "https://192.168.2.100:5000",
            "category": "云服务",
            "thumbnail": "..."
        }
    ]
}
```

### 🐛 Bug 修复

- 修复了设置界面存储信息显示不准确的问题
- 修复了天气功能在某些情况下的显示问题

## [1.0.0] - 2026-03-18

### ✨ 初始版本

#### 核心功能
- 链接管理：添加、编辑、删除链接
- 分类管理：创建、重命名、删除分类
- 搜索引擎：支持百度、Google、必应，可自定义
- 昼夜模式：支持浅色、深色、跟随系统
- 时间日期显示：支持多种格式和显示选项
- 天气显示：支持自定义城市和温度单位

#### 设置功能
- 链接管理标签页
- 外观设置标签页
- 显示设置标签页
- 搜索引擎管理标签页
- 数据管理标签页（导入/导出）
- 关于标签页

#### 界面特性
- 响应式设计
- 动画效果
- 缩略图预览
- 一键收起/展开分组
- 页面位置调整
